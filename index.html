I have created an interpreter for the FireFly programming language that is embedded into a browser using WebAssembly (WASM) technology. FireFly is a new interpreted programming language that takes inspiration from Python. While creating the interpreter, I primarily relied on the guidance of books and tutorials such as "Write an Interpreter in Go" and "Write a Compiler in Go" by Thorsten Ball and "Principles of Compiler Design" by A. Aho and J. Ullman. However, I also implemented a lot completely new features, including a classes, data model similar to Python's etc... There is still a lot of work to do in order to improve and add more features to the language, but I am doing it purely for the enjoyment of the process.
